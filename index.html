O que é o HTML

O HTML é a base de construção mais básico da Internet, é o que define o significado e toda a estrutura do contéudo da web. Junto do HTML são usadas algumas outras tecnologias, como por exemplo o CSS para estilizar a página do seu site ou o JavaScript que adiciona interatividade para seu site.

O HTML usa "Marcação" para anotar textos, imagens e outros tipos de conteúdos para serem exibidos em um navegador. Essas marcações são chamadas de "Tags". Utilizando essas "Tags" você consegue montar um arquivo HTML e com isso também montar uma página na web pra basicamente qualquer coisa, seja um blog, uma loja online ou até mesmo uma rede social (claro que você não vai conseguir montar tudo isso so com o HTML, ele será apenas sua base para montar a base de todo o seu projeto).

Arquivos Estáticos

São arquivos tipo imagem, JavaScript e CSS que tem a função de estilizar e disponibilizar interatividade com seu site, eles são chamados de estáticos pois o conteúdo desses arquivos é enviado exatamente como ele está no servidor, sem ter nenhuma mudança ou um processo adicional.

Esses arquivos não ficam na mesma pasta de templates HTML porque são recursos separados e que podem ser usados em várias paginas diferentes. Separando os arquivos estáticos dos templates nos conseguimos organizar melhor o nosso código, assim facilitando muito a manuntenção do projeto. Além de que essa separação permite que os arquivos estáticos sejam armazenados em um local específico para facilitar o gerenciamento.

Falando sobre Navegadores

Os navegadores são programas criados por empresas para que as pessoas que possuem Internet possam navegar por sites ou executar arquivos.

Navegadores normalmente seguem o padrão W3C que é o metódo de padronização que busca facilitar o desenvolvimento e interpretações de conteúdo para a Web.

Se deu inicío a criação de Navegadores com o "Mosaic", esse sendo lançado em 1993. Ele foi um dos tipicos Navegadores dos anos 90, foi um dos primeiros em disponibilizar interface gráfica multimídia para a grande variedade de serviços de informação da Rede, em uma época onde o domínio da Internet estava se expandindo rapidamente para fora das Universidades e Instituições Industriais.

Caso precise saber se o navegador que pretende usar tem compatibilidade com certas partes do HTML você pode clicar no link que ficará disponivel no final do blog.

Tipos de Servidores

Servidor é um software ou computador, com sistemas de computação focada em fornecer serviços a uma rede de computador chamada de "cliente".

Esses serviços podem ser de tipos variados, indo de arquivos até mesmo e-mails.

Os tipos de Servidores mais comuns são aqueles conhecidos como cliente-servidor, é usada em redes de médio e grande porte (com várias máquinas) e em redes onde a questão de segurança tem grande importância. O termo "servidor" é aplamente aplicado a computadores completos, porém um servidor pode equivaler a um software ou a partes de um sistema computacional, ou até mesmo em uma máquina que não seja um computador.

Mais alguns exemplos de servidores:

  • Proxy: É basicamente um filtro, bloqueia algo que não pode ser acessado (Muitas vezes usado em empresas para limitar o acesso dos funcionários em certos sites).
  • Firewall: Barreira de segurança para acessos externos.
  • Intranet: Servidor Web que pode ser acessado sem estar necessariamente conectado a "Internet", mas sim com a rede local que o servidor disponibiliza.
  • DNS:É o que liga o nome da pesquisa feita com o IP do site que o cliente esta buscando (basicamente uma lista telefônica).

Hospedagem de Sites

Nesse meio de serviços de criação de sites você pode ter o seu proprio server para alocar seus arquivos para disponibilizar para o cliente depois. Ou contratar uma empresa para hospedar arquivos(ou e-mails) do seu site.

Várias dessas empresas dispnibilizam um domínio para seu site, fazendo com que a busca por ele seja mais fácil.

Conexão FTP

FTP pode ser executado em modo ativo ou passivo, os quais determinam como a conexão de dados é estabelecida. No modo ativo, o cliente envia para o servidor o endereço IP e o número da porta na qual ele irá ouvir e então o servidor inicia a conexão TCP. Em situações onde o cliente está atrás de um firewall e inapto para aceitar entradas de conexões TCP, o modo passivo pode ser utilizado. O cliente envia um comando PASV para o servidor e recebe um endereço IP e um número de porta como resposta, os quais o cliente utiliza para abrir a conexão de dados com o servidor.[1] Ambos os modos foram atualizados em Setembro de 1998 para adicionar suporte ao IPv6 e feitas algumas mudanças no modo passivo, tornando-o modo passivo estendido.

Conexão FTP é a conexão feita diretamente com o servidor para o envio de arquivos do seu site, para alocar eles no próprio servidor.

Usando a conexão FTP e o programa Filezilla você consegue passar seus arquivos HTML para o servidor, e assim seu site finalmente começa a funcionar para o uso público.

A tranSferência de dados pode ser feita em qualquer um dos três modos a seguir:

  • Modo fluxo: dado é enviado como um fluxo contínuo, liberando FTP de fazer algum processamento. No lugar disso, todos o processamentos é deixado para o TCP. Nenhum indicador de fim de arquivo é necessário, a menos que o dado esteja divido dentro de registros.

  • Modo de bloco: o FTP quebra o dado dentro de vários blocos (bloco de cabeçalho, contagem de byte e campo de dado) e então passa para o TCP.

  • Modo comprimido: dado é comprimido utilizando um algoritmo simples.
